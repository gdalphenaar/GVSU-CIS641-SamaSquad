what is this project about?

project sama is centered around building a guitar temperature and humidity monitoring system using a raspberry pi as the "base unit" along with common environmental sensors (bosch bme280) to monitor (very) local conditions. this system will also have a web interface allowing users to monitor environmental conditions in (semi) real-time and set appropriate "safe" levels for both temperature and humidity. for more detailed information, visit the info page. for random updates as the project progresses, see the updates page.

why this project?

lots of other guitar temperature and humidity monitors exist - this is true. however, many of the existing off-the-shelf sensors are proprietary and tied to specific apps and services that often need accounts and whatnot. a lot of them are also often expensive - approaching $50 per unit. therefore, the purpose of this project is to do the opposite - create an open guitar temperature and humidity monitoring system based on (relatively) inexpensive commercially-available components that doesn't need to be linked to any account.

what technologies are we using?

raspberry pi microcomputer
bosch bme280 temperature / humidity sensor
wireless transmitter board
rechargeable battery & power delivery board
